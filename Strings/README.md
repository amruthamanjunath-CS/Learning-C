## String Programs
This folder contains programs demonstrating string operations in C.

1. **String Concatenation**  
   - Concatenates two strings **without using built-in string functions**.

2. **Menu-driven String Program**  
   - Provides a menu for the user to perform different operations on strings:  
     - **Palindrome Check** – Checks if a string is a palindrome.  
     - **String Length** – Calculates the length of the string.  
     - **Reverse String** – Reverses the given string.
