# Mini Projects

This folder contains several mini projects implemented in C.

## 1. Booking System
A bus ticket booking system that allows you to:
- **Book a bus ticket** – Reserve a seat on a bus.
- **Cancel a bus ticket** – Cancel a previously booked ticket.
- **Exit** – Exit the application.

## 2. Password
A password security checker that ensures strong passwords. Features include:
- Checking for **uppercase letters**.
- Checking for **lowercase letters**.
- Checking for **numbers**.
- Checking for **special characters**.
- Suggesting improvements if the password does not meet the requirements.

## 3. Struct (Student Data Analysis)
A program to manage student data using structs. Features include:
- **Add a student** – Insert a new student record.
- **Update a student** – Modify existing student details.
- **Remove a student** – Delete a student record.
- **View all students** – Display all student records at once.
