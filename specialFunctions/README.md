## Special Functions
This folder contains programs demonstrating special built-in functions or features in C.

1. **ASCII to Integer (ATOI)**  
   - Converts a string to an integer using the standard library function `atoi()`.

2. **Command-line Sum**  
   - Takes numbers as input from the command line during execution.  
   - Uses **command-line arguments (`argc` and `argv`)** and arrays/vectors to calculate the sum of the numbers.
