# Functions & Recursions

This folder contains programs implemented in C using functions and recursion.

## Recursion Programs
1. **Sum using Recursion** – Calculates the sum of `n` numbers using recursion.
2. **Factorial using Recursion** – Calculates the factorial of a number using recursion.
3. **Fibonacci using Recursion** – Generates the Fibonacci series up to n terms using recursion.
4. **Palindrome using Recursion** – Checks if a given string or number is a palindrome using recursion.

## Function Programs
1. **Natural Number to Roman Numeral Conversion** – Converts a given natural number to its Roman numeral equivalent using functions.
